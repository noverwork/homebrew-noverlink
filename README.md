# Homebrew Tap for Noverlink

Official Homebrew tap for [Noverlink](https://github.com/noverwork/noverlink) - a local-to-global tunneling CLI.

## Installation

```bash
brew tap noverwork/noverlink
brew install noverlink
```

## Usage

```bash
# Login to your account
noverlink auth login

# Start a tunnel
noverlink http 3000
```
